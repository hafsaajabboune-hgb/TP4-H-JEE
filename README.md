TP - Hibernate & JPA avec MySQL

Objectif
Configurer une application Java avec Hibernate et MySQL pour la persistance des données.

Étapes réalisées

- Configuration Maven (dépendances Hibernate, JPA, MySQL)

- Création des entités JPA

- Développement des services

- Tests de persistance

# Résultat
Application fonctionnelle avec gestion complète de la persistance via Hibernate et MySQL.
